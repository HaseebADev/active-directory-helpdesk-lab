# Active Directory Helpdesk Lab

## Overview
This project demonstrates a hands-on Active Directory helpdesk lab designed to simulate real-world first-line IT Support and Service Desk tasks in a business environment.

The lab focuses on common support activities such as user account management, access troubleshooting, and incident resolution using Active Directory.

## Lab Environment
- VMware Workstation Pro
- Windows Server (Domain Controller)
- Windows 10/11 Client Machine
- Active Directory Domain Services (AD DS)
- Remote Desktop Protocol (RDP)

## Helpdesk Tasks Completed
- Created and managed user accounts in Active Directory
- Reset passwords and unlocked locked user accounts
- Created security groups and managed group membership
- Troubleshot login and access issues caused by incorrect permissions
- Simulated common Service Desk incidents and resolutions

## Example Helpdesk Scenarios
- User unable to log in due to account lockout
- Password reset requests
- Missing access due to incorrect group membership
- Basic workstation access troubleshooting

## Documentation & Support Process
- Followed ITIL-aligned first-line support workflows
- Logged issues, actions taken, and resolutions
- Identified issues requiring escalation to higher support levels
- Created clear documentation to support repeat issue resolution

## What This Lab Demonstrates
- Practical Active Directory fundamentals
- Understanding of first-line IT Support responsibilities
- Ability to troubleshoot user access and authentication issues
- Strong documentation and communication skills
- Experience working in a virtualised environment using VMware
